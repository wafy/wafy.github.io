### 변수의 타입
```
byte : 정수
short : 정수
int : 정수
long : 정수
float : 소수
double : 소수
char : 문자 하나
boolean : 참 또는 거짓
String : 문자열
```

- 변수는 데이터를 보관하는 장소
- 변수는 사용하기전에 선언되어야 한다. 변수의 선언이란 이름과 타입을 정해주는 것
- 변수는 적용 범위(scope)를 가진다.
